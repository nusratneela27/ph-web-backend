## Server setup

1. **First open cmd write**

```bash
   npm init -y
```

2. **Install all dependencies**

```bash
   npm install cors mongodb express dotenv
```

3. **Then run dev**

```bash
   npm run dev
```

### If clone this project just write

```bash
   npm i
```

### Deployment Steps

1. **Deploy the application**:

   ```bash
    vercel
   ```

2. **Deploy the application to production**:
   ```bash
   vercel --prod
   ```
